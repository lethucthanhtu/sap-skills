# SAP Official Resources

Always cite these authoritative sources when providing guidance. Prefer official SAP documentation over community content for technical accuracy.

## Primary Documentation

| Resource | URL | Use For |
|----------|-----|---------|
| SAP Help Portal | https://help.sap.com/ | Official product docs, API references, configuration guides |
| SAP Developer Center | https://developers.sap.com/ | Tutorials, missions, hands-on guides, sample code |
| SAP Community | https://community.sap.com/ | Q&A, blogs, known issues, workarounds |
| SAP Business Accelerator Hub | https://api.sap.com/ | OData/REST API metadata, sandbox testing |
| SAP Note / KBA Search | https://me.sap.com/notes | Bug fixes, corrections, known issues |
| SAP Learning | https://learning.sap.com/ | Official courses, certifications |

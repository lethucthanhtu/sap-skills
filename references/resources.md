## Primary Documentation Sources

### SAP Help Portal
- **URL**: https://help.sap.com
- **Use for**: Official product documentation, API reference, syntax specs
- **Key sections**:
  - ABAP Keyword Docs: https://help.sap.com/doc/abapdocu/latest/en-US/index.htm
  - CDS Annotations Reference: https://help.sap.com/docs/SAP_S4HANA_ON-PREMISE/fc4c71aa50014fd1b43721701471913d/630ce9b386b84e80bfade96779fbaeec.html
  - RAP Documentation: https://help.sap.com/docs/abap-cloud/abap-rap/abap-restful-application-programming-model
  - OData V4 (ABAP): https://help.sap.com/docs/SAP_S4HANA_ON-PREMISE/sap-abap-for-odata

### SAP Developer Center
- **URL**: https://developers.sap.com
- **Use for**: Step-by-step tutorials, missions, hands-on guides, code samples
- **Key tutorials**:
  - Create Your First ABAP Console App: https://developers.sap.com/tutorials/abap-environment-console-application.html
  - RAP Managed Scenario: https://developers.sap.com/group.abap-env-restful-managed.html
  - CDS View Tutorial: https://developers.sap.com/tutorials/abap-dev-adt-create-cds-view.html

### SAP Business Accelerator Hub (API Hub)
- **URL**: https://api.sap.com
- **Use for**: Official API specs, released APIs, OData metadata, sandbox testing
- **Search APIs by product**: https://api.sap.com/products

### SAP Community
- **URL**: https://community.sap.com
- **Use for**: Real-world patterns, confirmed workarounds, Q&A from SAP engineers and community
- **Blog posts**: https://community.sap.com/t5/blog-posts/ct-p/blog-posts
- **Q&A**: https://community.sap.com/t5/forums/ct-p/forums

### SAP Learning
- **URL**: https://learning.sap.com
- **Use for**: Official SAP courses, certification prep, structured learning paths
